# わんワード - CLAUDE.md

## プロジェクトの目的

犬用語のカタカナ4文字を当てる、Wordle風の1日1問パズルゲーム「わんワード」。HTML/CSS/JSのみで動く静的サイトとして実装し、GitHub Pagesで公開する。

## 技術スタック

- HTML5 / CSS3 / Vanilla JavaScript（ビルドツール・フレームワーク不使用）
- GitHub Pages（ホスティング）

## ディレクトリ構成の方針

- `index.html`: エントリーポイント
- `css/`: スタイル
- `js/`: ゲームロジック（単語リスト、日替わり選出、判定処理、UI制御）
- 単語データは外部APIを使わず、JS内の静的配列で保持する想定

## コーディング規約

- ES2015+のモダンな素のJavaScriptを使用し、外部依存を増やさない
- インデントは半角スペース2つ（`.editorconfig` 準拠）
- 犬用語の単語リストはネタバレ防止のため、コンソールログなどに平文で出力しない

## よく使うコマンド

- ローカル確認: `python -m http.server 8000` などの簡易サーバーで `index.html` を配信して確認
- Lint/テストの仕組みは未整備。ロジックを切り出す場合は `tests/` に単体テストを配置する

## 注意事項

- `.env` は読み取らない・コミットしない
- 単語リストを追加・変更する際は、その日の出題が過去の出題と重複しないか確認する

※ コードが揃ったら `/init` で本ファイルを更新・拡充できます。
